
# Tripverse App

Projeto de UI/UX de um aplicativo de viagens desenvolvido no Figma.

# Sobre
O Tripverse é um aplicativo que permite explorar destinos, visualizar detalhes e realizar reservas de forma simples.

# Telas
- Splash
- Login
- Criar conta
- Home
- Explorar
- Detalhes do destino
- Reserva
- Confirmação
